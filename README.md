# Scientific Prototypes

A collection of applied research prototypes demonstrating practical applications of machine learning, signal processing, and AI technologies.

## Projects

### 1. [Timeseries Toolkit](./timeseries-toolkit/)

An **Adaptive AutoML Framework** for financial time series analysis with automatic regime detection, pipeline selection, and explainable forecasting.

**Key Features:**
- Automatic market regime detection (bull/bear/crisis/sideways)
- Self-configuring analysis pipelines based on data characteristics
- Multiple forecasting models: Kalman Filter, LightGBM, Ensemble
- Causal analysis (CCM, Granger causality)
- Comprehensive diagnostics and validation
- Explainable reports in Markdown format
- Data sources: Crypto, Equities, Rates, Liquidity, Volatility, Macro (via free APIs)

**Status:** Complete (311 tests passing)

[View Project](./timeseries-toolkit/)
